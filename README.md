🌡️ Temperature Converter

The Temperature Converter is a simple Java application that converts temperature values between Celsius, Fahrenheit, and Kelvin. It provides a menu-driven interface where users can select a conversion type, enter a temperature value, and receive accurate results instantly.

This project demonstrates the use of core Java concepts such as user input handling, switch-case statements, and mathematical calculations. It is designed to be easy to use and helps users perform temperature conversions efficiently.

Features
Celsius ↔ Fahrenheit
Celsius ↔ Kelvin
Fahrenheit ↔ Kelvin
Interactive console-based interface
Accurate conversion calculations
Technologies Used
Java
Scanner Class
Switch Case Statements
